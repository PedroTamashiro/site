Estrutura básica do HTML

Voltar

Assim como tudo na vida, o estudo de HTML tamebém há um inicio, e é aqui onde começamos, para base, é importante saber a estrutura de qualquer código de HTML com isso, o código é iniciado com html, para definir o programa como um programa web.

Seguindo o HTML temos 2 outras estruturas, head: aqui seria o titulo principal da pagina, para o desenvolvimento, este é todo o conteudo que sera baixado antes da visualização do usuário, além dele, temos o body: Nele é todo o corpo do texto em si, o que o usuário vai realmmente ver e poderá interagir.

Atributos das TAG

Voltar

Conceito

Nesta aula nos é introduzido a ideia de dar caracteristicas para determinados objetos na página, sendo divididos em 2 tipos, os gerais e os específicos.

Gerais

Id

O id é um atributo utilizado na programação para referenciar determinado objeto dentro do código, como no uso dos links para voltar ou avançar a visualização, como a usada nesta página.

Style

O style (Estilo), ele aplica uma configuração de CSS no objeto, alterando as caracteristicas de exibição do elemento, podendo mudar de cor, forma ou oque for.

Exemplo mudando de cor

Class

Classes são padrões que podem ser aplicadas a qualquer momento dentro do código, como uma receita pronta a ser seguida pelo computador, agilizando muito a tarefa do programador.

Específicos

type

Para a aplicação do Type, é necessário compreender o uso do input, que é uma forma de entrada dentro do site, sendo utilizada pelo usuário para inserir informações uteis para o sitema.

Ai que entra o Type, o type é utilizado justamente para modificar esse tipo de entrada, limitando ela para o usuário, podendo ser do tipo numero, texto, email ou até mesmo telefone.

Src

Esse atributo tem uma unica função, incluir links nas Tags do tipo imagem, puxando imagens direto da internet por meio deles.
